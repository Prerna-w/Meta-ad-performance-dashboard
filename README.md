# Meta-ad-performance-dashboard
Meta Ad Performance Dashboard analyzing campaign KPIs (CTR, engagement, conversions) to identify funnel drop-offs and optimize marketing ROI.

Project Overview

This project focuses on analyzing the performance of Meta (Facebook/Instagram) ad campaigns using data-driven techniques.
The goal is to evaluate campaign effectiveness across the marketing funnel and identify opportunities to improve conversions and ROI.

The dashboard provides insights into impressions, clicks, engagement, conversions, audience behavior, and ad performance using interactive visualizations built in Power BI.

🎯 Objectives

Analyze end-to-end marketing funnel performance

Identify drop-offs between awareness, engagement, and conversion stages

Evaluate campaign performance across demographics and regions

Optimize ad strategy using data-driven insights

📌 Key Metrics Analyzed

Impressions: 216,000

Clicks: 25,400

CTR (Click Through Rate): 11.76%

Engagement Rate: 13.56%

Purchases: 1,300

Conversion Rate (Clicks → Purchase): 5.21%

Purchase Rate (Impressions → Purchase): 0.61%

📊 Dashboard Features

📈 Funnel Analysis (Awareness → Engagement → Conversion)

👥 Audience Segmentation (Age, Gender)

🌍 Geographic Performance Analysis

⏰ Time-based Insights (Hourly & Daily trends)

🎥 Ad Format Performance (Video, Stories, Image, Carousel)

💰 KPI Tracking (CTR, Conversion Rate, Engagement Rate)

🔍 Key Insights

High CTR (11.76%) indicates strong ad creatives and targeting

Strong engagement rate (13.56%) shows effective audience interaction

Significant drop in conversion (0.61%) suggests funnel inefficiency

Female users (18–30 age group) contribute the highest engagement

India & Brazil drive high traffic volume, while Germany & UK show higher purchase potential

Video and Story ads outperform other formats in engagement and conversions

Peak engagement observed during afternoon and evening hours
